Object Oriented Programming Programming Language

What is Java?

Java is a programming language that implements the paradigm of object oriented programming.

It was originally developped to be run independently from the OS.

It is very powefull and used in many domains:

  • Desktop applications
  • Mobile applications
  • Back-end of web applications
To find more information about this amazing programming language, please visit and learn more about it.

Classes and Objects

Java language is based on classes and objects and their relationship. But what are they?

Classes

Classes are abstractions that we can implement to simulate the standard behavior of an object. It is a standard pattern of a abstract idea.

Objects

Objects are instances of a class. They have their own properties and attributes. Besides that, objects are unique and are a concrete type of a class.
